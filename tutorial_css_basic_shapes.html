Squares, Triangles and Circles

Click to see how to do:

First of all, imagine a square in your mind. Something like the down below, huh?

Two horizontal lines plus two vertical lines. So we can do it:
.square{
width: 100px;
height: 100px;
background-color: blueviolet;
}

So, a rectangle is just like two squares together:

Two horizontal lines plus two vertical lines. So we can do it:
.rectangle{
width: 200px;
height: 100px;
background-color: blueviolet;
}

A triangle is a little bit harder, but isn't a seven head beast.

So we can do it:
.triangle{
width: 0;
height: 0;
border-left: 50px solid transparent;
border-right: 50px solid transparent;
border-bottom: 100px solid blueviolet;
}

A circle is a square that round to much.

Catch ya square, and round it using border-radius:


.circle{
width: 100px;
height: 100px;
border-radius: 50%;
background-color: blueviolet;
}
